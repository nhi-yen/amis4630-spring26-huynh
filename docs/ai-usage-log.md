## CartController
- **Generated by:** Copilot  
- **Issues in AI output:**  
  - Missing Include/ThenInclude for Product.  
  - Missing upsert logic.  
  - Missing ownership checks.  
- **Modifications I made:**  
  - Added hardcoded user ID (`default-user`).  
  - GET /api/cart  
    - Uses `.Include(c => c.Items).ThenInclude(i => i.Product)`  
    - Returns **404** when no cart exists  
    - Maps to CartResponse with computed totals  
  - POST /api/cart  
    - Validates product exists  
    - Implements upsert (increment quantity if product already in cart)  
    - Returns **201 Created**  
  - PUT /api/cart/{cartItemId}  
    - Validates ownership  
    - Returns **200 OK**  
  - DELETE /api/cart/{cartItemId}  
    - Ownership check  
    - Returns **204 No Content**  
  - DELETE /api/cart/clear  
    - Removes all items  
    - Returns **204 No Content**
